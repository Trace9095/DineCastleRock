## Recommended Image Dimensions

For optimal display on the website:
- **Card thumbnails:** 800 x 600px (4:3 ratio)
- **Hero images:** 1920 x 1080px (16:9 ratio)
- **Gallery images:** 1200 x 800px

## File Naming Convention

Use this format: `[business-slug].jpg`
Example: `scileppis.jpg`, `castle-cafe.jpg`, `great-divide.jpg`

Save all images to: `/public/images/listings/`
